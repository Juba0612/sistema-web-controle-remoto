# Arquivos utilizados

Nesta etapa do projeto, foi necessário o desenvolvimento de um [sistema web](PI.zip) para a autenticação dos usuários e controle de atividades de acordo com sua autorização no sistema, uma [dashboard no NodeRED](flows.json) para lidar com o fluxo dos dados, exibição e controle destes, além da [atualização do banco de dados](ltr_control.sql) utilizado na [etapa anterior do projeto](../atividade-2/ltr_control_equipe4.sql). Para a simulação do envio de dados, foi utilizado uma nova coleção desenvolvida no [Postman](https://matheus-v-barbosa-7651800.postman.co/workspace/Matheus-Ventura-Barbosa's-Works~b76e2801-31f1-48c1-8c66-55af14d4ebe8/collection/50028470-77de773c-b1f1-459e-b510-d428e3829b47?action=share&creator=50028470).
